# Pet Images

Place one clear PNG or SVG image for each pet in this folder.

## Naming Convention
- Use lowercase, hyphen-separated names (e.g., `cave_hyena.png`, `arctic_wolf.svg`)
- Recommended size: 256x256px or SVG
