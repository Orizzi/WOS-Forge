# Pet Item Images

Place icons for all pet upgrade items in this folder.

## Naming Convention
- Use lowercase, hyphen-separated names (e.g., `pet-food.svg`, `evolution-stone.svg`)
- SVG preferred for scalability
- Recommended size: 64x64px or SVG

## Required Images
- pet-food.svg
- manual.svg
- potion.svg
- serum.svg
- evolution-stone.svg
- advanced-stone.svg
- premium-stone.svg
