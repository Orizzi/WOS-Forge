# Pets Calculator Prep Checklist

Use this checklist to ensure everything is ready before building the pets calculator page.

---

## 1. Data Structure
- [ ] Excel sheet covers all pets, levels, resources, milestones, skills
- [ ] CSV format standardized and ready for extraction
- [ ] All columns named consistently (Pet_Name, Level, Food, Manual, Potion, Serum, SVS_Points, Evolution_Stones, Adv_Stones, Prem_Stones, etc.)
- [ ] Data filled for all pets and levels

## 2. Assets
- [ ] All pet images (PNG/SVG) named and placed in `src/assets/resources/pets/`
- [ ] All item/resource icons (SVG preferred) in `src/assets/resources/pet-items/`
- [ ] Alt text prepared for accessibility

## 3. UI/UX Design
- [ ] Page layout sketched or described (pet selector, level inputs, inventory, results, skill info)
- [ ] Accessibility features planned (ARIA labels, keyboard navigation)
- [ ] Responsive design planned (mobile/tablet/desktop)

## 4. Calculator Logic
- [ ] List of all calculations needed (resources, milestones, skills, power/stat gains)
- [ ] Error handling strategy (invalid ranges, missing data)
- [ ] Inventory comparison logic (deficit/surplus)

## 5. Translations
- [ ] List of new translation keys (pet names, item names, UI labels, error messages)
- [ ] Ready to add to `translations.js` for all supported languages

## 6. Integration Points
- [ ] Unified profile system integration planned
- [ ] Batch operations, export/import, profile sharing considered

## 7. Testing
- [ ] Manual test cases prepared (edge cases, milestones, inventory checks)
- [ ] Automated testing plan (if applicable)

## 8. Documentation
- [ ] Quick README or doc section drafted for users/contributors
- [ ] Usage instructions and data update workflow described

---

**Complete this checklist before starting page implementation for a smooth, robust, and user-friendly calculator!**
