# Archive Folder

This folder contains files and directories that are not part of the active WOS Calculator project but are kept for reference or potential future use.

## Contents

### `/tmp-folders/`
Contains temporary/external tools and bots that were not part of the main calculator:
- `tmp-wos-bot/` - External bot tool
- `tmp-wos-gift/` - Gift code redemption tool
- `tmp-wos-gift-redeem/` - Another gift code tool variant
- `tmp-wrs/` - WRS related scripts

### `/root-legacy-docs/`
Contains duplicate or legacy documentation that was previously at root level but is now redundant (proper docs are in `/docs/`):
- Architecture documents (TECHNICAL_ARCHITECTURE.md, MODULE_GRAPH.md, etc.)
- System guides (I18N_SYSTEM.md, THEME_SYSTEM.md, UI_FLOW.md, etc.)
- Development guides (DEVELOPER_GUIDE.md, AI_AGENT_GUIDE.md, etc.)
- Legacy utility scripts (list-sheets.js, verify-fc-totals.js)
- Draft/alternate READMEs
- Images and guides

### Other Archived Items
- `Projet/` - Nested/duplicate project folder
- `research-icons/` - Icon research and scraping tools
- `gift-code-service/` - Gift code service implementation

## Purpose

These files are archived to keep the root directory clean and focused on the active project while preserving potentially useful reference material or tools that might be needed in the future.

## Usage

If you need any of these files:
1. Review the contents to understand what was archived
2. Copy (don't move) back to the appropriate location if needed
3. Update any file references in the main project
